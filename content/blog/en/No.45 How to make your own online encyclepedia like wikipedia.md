# How to make your own online encyclopedia like Wikipedia

Wikipedia is the most well-known online encyclopedia in the world, and you could set up an encyclopedia similar to it which could be visited by everyone on the Internet.

MediaWiki is a FOSS software used successfully on Wikipedia and lots of sister projects. The project is based on PHP and MySQL which is easy to set up. 

The procession of setting up MediaWiki straightly is very verbose. To simplify this process, you could set up your own MediaWiki using docker.
