# How to set up my blog on GitHub pages

GitHub pages is a great way for hosting blogs.

To make use of GitHub pages, you should have a GitHub account.

1. Set up a new repository.

2. Set up your local blog using hexos or mkdocs.

3. Download your repository with the Git clone command.

4. Move the content in your blog folder to the local repository folder.

5. Push your repository.

So far, your blog is uploaded to GitHub pages, view it on browser with the link "https://www.github.io/your account name/your repository name"
